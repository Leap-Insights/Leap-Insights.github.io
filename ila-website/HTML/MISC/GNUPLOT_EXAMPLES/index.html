Making graphs using Gnuplot

Gnuplot makes good quality graphs and is simple to use. It has an easy to use help. It has both a unix and a windows version available free on the net. It can be used both interactively and with batch files.

Gnuplot generates graphs in .eps files. The resulting pictures are of high quality. These graphs can be converted into .pdf files with the command: $ epstopdf "filename.eps".
